# 🫀 CardioAI – Heart Disease Detection System

CardioAI is an AI-powered web application that predicts heart-related diseases based on user symptoms using machine learning. The system provides real-time predictions, explainability using LIME, doctor recommendations, and downloadable health reports.

---

## 🚀 Features

- 🔮 Heart Disease Prediction (Multi-class classification)
- 📊 Confidence Score with interactive visualization
- 🔬 LIME Explainability (feature contribution)
- 👨‍⚕️ Doctor Recommendation based on city
- 📅 Appointment Booking System
- 📄 Downloadable PDF Health Report
- 📈 Model Insights (Confusion Matrix, Classification Report)
- 🎨 Modern Streamlit UI Dashboard

---

## 🤖 Machine Learning Models

- 🌲 Random Forest (tuned using GridSearchCV)
- 🧮 Support Vector Machine (SVM)
- 🗳️ Voting Classifier (Ensemble Model)

---

## 📂 Dataset

- Source: Kaggle Diseases Dataset  
- Size: ~1.5 lakh records  
- Original Features: 150+  
- Selected Features: 13 (after preprocessing & feature selection)

---

## 🧪 Model Performance

- Accuracy: 97%
- Precision: 96%
- Recall: 95%
- F1 Score: 95%
- ROC-AUC: 99%

> Note: Metrics are precomputed from the trained model and verified in the Jupyter Notebook.

---

## 🧠 Explainability

LIME (Local Interpretable Model-Agnostic Explanations) is used to explain predictions by showing how each symptom contributes to the final decision.
